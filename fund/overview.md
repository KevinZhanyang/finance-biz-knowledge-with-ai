# 基金行业概览

## 什么是基金

基金（证券投资基金）是一种集合投资工具：投资者把钱交给专业的基金管理公司，由基金经理按照约定策略进行投资，收益和风险由投资者承担。

## 基金分类

### 按募集方式
- **公募基金**：面向公众募集，门槛低（1元起），受证监会严格监管，信息披露要求高
- **私募基金**：面向合格投资者（100万起），灵活度高，信息披露要求较低

### 按投资标的
| 类型 | 主要投资方向 | 风险等级 |
|------|-------------|---------|
| 货币基金 | 短期货币工具（存单、国债逆回购） | 低 |
| 债券基金 | 债券（国债、企业债、可转债） | 中低 |
| 混合基金 | 股票+债券灵活配置 | 中 |
| 股票基金 | 股票（持仓≥80%） | 中高 |
| 指数基金 | 跟踪特定指数 | 中高 |
| QDII基金 | 境外市场 | 中高 |
| FOF基金 | 其他基金 | 中 |

### 按交易方式
- **开放式基金**：随时申购赎回（T+1确认）
- **封闭式基金**：封闭期内不可赎回，可在二级市场交易
- **ETF**：交易所交易基金，可在场内买卖，也可一级市场申赎（实物申赎）
- **LOF**：上市开放式基金，既可场内买卖也可场外申赎

## 基金生命周期

```
产品设计 → 证监会审批 → 募集发行（认购期）→ 成立（验资）→ 建仓期 → 正常运作（申购/赎回）→ 清盘
```

### 关键节点
1. **认购期**：基金发行阶段，投资者以1元面值认购，通常2-4周
2. **成立**：募集金额和人数达到要求后基金成立
3. **建仓期**：基金经理建立初始投资组合，通常不超过3个月
4. **开放申赎**：进入正常运作，投资者可自由申购赎回
5. **清盘**：基金资产净值过低或持有人过少时触发清盘
