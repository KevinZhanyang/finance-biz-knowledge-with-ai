# 基金代销

## 什么是代销

基金代销是指销售机构（银行、券商、第三方平台）受基金公司委托，向投资者销售基金产品。代销机构不拥有基金产品，只提供销售渠道。

## 核心参与方

```
投资者 ←→ 代销机构 ←→ 基金公司（含TA系统）←→ 托管银行
                                                  ↕
                                              中国结算
```

- **投资者**：购买基金的终端客户
- **代销机构**：提供销售渠道、客户服务、适当性管理
- **基金公司**：产品管理、投资运作、份额登记（通过TA系统）
- **托管银行**：资金保管、资金清算、净值复核

## 代销渠道类型

### 银行代销
- 最大的代销渠道，占公募基金销售额的大头
- 优势：网点多、客户基数大、客户信任度高
- 特点：偏好销售新基金（认购费收入），尾随佣金是重要收入来源

### 券商代销
- 通过证券账户销售基金
- 优势：客户投资经验较丰富，场内基金（ETF）的主要渠道
- 特点：与股票交易使用同一账户体系

### 独立销售机构（第三方）
- 如天天基金（东方财富）、蚂蚁基金、好买基金等
- 优势：费率折扣大（通常1折）、产品覆盖广、比价方便
- 特点：互联网化程度高，用户体验好

### 直销
- 基金公司自有渠道（官网、APP、直销柜台）
- 优势：费率可能有优惠，机构客户通常走直销
- 参见 [直销](./direct-sales.md)

## 代销业务流程

### 1. 签署代销协议
基金公司与代销机构签署《基金代销协议》，约定：
- 代销产品范围
- 费率及佣金分成（尾随佣金比例）
- 数据交换方式和时间
- 客户服务责任划分

### 2. 产品上架
- 代销机构在自身平台上架基金产品
- 配置产品信息（风险等级、费率、限额等）
- 进行投资者适当性匹配

### 3. 交易受理
```
投资者下单 → 代销机构受理 → 生成申请数据 → 发送给基金公司TA系统
```
- 代销机构在交易时间内受理投资者的申购/赎回请求
- 交易时间：交易日 9:30-15:00（15:00后的申请算下一交易日）
- 代销机构汇总当日申请数据，通过接口文件发送给基金公司

### 4. 资金清算
- **申购**：代销机构归集投资者资金 → 划付至基金托管账户
- **赎回**：基金托管账户 → 划付至代销机构 → 代销机构发放给投资者
- 清算通过银行间支付系统完成

### 5. 份额确认
- 基金公司TA系统根据当日净值计算份额
- T日申购：T日净值确认，T+1日可查询份额
- 确认结果通过接口文件回传给代销机构

## 收入模式

### 代销机构的收入来源
1. **申购费分成**：投资者支付的申购费，代销机构通常拿大头
2. **尾随佣金（客户维护费）**：基金公司从管理费中按比例支付给代销机构，按保有量计算
3. **赎回费分成**：部分赎回费归代销机构

### 尾随佣金
- 这是代销机构最重要的持续性收入
- 计算方式：`日尾随佣金 = 当日保有量 × 年费率 / 365`
- 费率通常在 0.2%-0.5% 之间，由基金公司与代销机构协商

## 数据交换

代销机构与基金公司之间的数据交换是代销业务的核心技术环节：

- **接口文件**：通常使用TXT定长/变长格式，遵循行业标准（如中登格式、开放式基金业务数据交换协议）
- **交换时间**：日终批量处理，每日固定时间点交换文件
- **交换内容**：交易申请、确认结果、份额对账、分红数据等
