# 基金术语表

## 交易相关

| 术语 | 英文 | 含义 |
|------|------|------|
| 认购 | Subscription | 基金募集期间购买基金，按面值(1元)购买 |
| 申购 | Purchase | 基金成立后日常购买基金，按当日净值购买 |
| 赎回 | Redemption | 卖出基金份额，按当日净值计算 |
| 定投 | Regular Investment | 定期定额投资基金 |
| 转换 | Conversion/Switch | 将持有的基金份额转为同一基金公司的另一只基金 |
| 转托管 | Transfer of Custody | 将基金份额从一个代销机构转到另一个代销机构 |
| 巨额赎回 | Massive Redemption | 单日赎回申请超过基金总份额10%，基金公司可延期办理 |
| 强制赎回 | Forced Redemption | 因违规等原因被强制赎回份额 |

## 净值相关

| 术语 | 英文 | 含义 |
|------|------|------|
| 单位净值 | NAV (Net Asset Value) | 基金每份的价值，= 基金净资产 / 总份额 |
| 累计净值 | Cumulative NAV | 考虑历史分红后的单位净值，反映基金成立以来的总收益 |
| 估值 | Valuation | 计算基金持有资产的公允价值 |
| 估值表 | Valuation Sheet | 基金每日的资产明细及净值数据 |

## 费率相关

| 术语 | 英文 | 含义 |
|------|------|------|
| 申购费 | Purchase Fee | 购买基金时支付的手续费，通常按申购金额的百分比收取 |
| 赎回费 | Redemption Fee | 赎回基金时支付的费用，持有时间越长费率越低 |
| 管理费 | Management Fee | 基金公司收取的管理报酬，按基金净值的年化百分比每日计提 |
| 托管费 | Custody Fee | 托管银行收取的保管费，按基金净值的年化百分比每日计提 |
| 销售服务费 | Distribution Fee | C类份额收取，替代申购费，按日计提 |
| 尾随佣金 | Trailer Fee | 基金公司从管理费中支付给代销机构的客户维护费 |

## 运营相关

| 术语 | 英文 | 含义 |
|------|------|------|
| 日终 | End of Day (EOD) | 每日收市后的批量数据处理流程 |
| 跑批 | Batch Processing | 系统集中批量处理业务数据 |
| 清算 | Clearing | 计算交易各方应收应付金额 |
| 交收 | Settlement | 实际完成资金和证券的交割 |
| TA | Transfer Agent | 注册登记机构/系统，负责份额登记 |
| 份额确认 | Share Confirmation | TA系统确认投资者的申购/赎回份额 |
| 对账 | Reconciliation | 各机构间核对数据一致性 |
| 分红 | Dividend | 基金将收益分配给持有人 |
| 红利再投资 | Dividend Reinvestment | 将分红金额自动再购买基金份额 |

## 时间相关

| 术语 | 含义 |
|------|------|
| T日 | 交易日，指投资者提交申请的当天（15:00前） |
| T+1 | 下一个交易日 |
| T+N | 第N个交易日后 |
| 交易日 | 证券交易所的工作日（排除周末和法定节假日） |
| 开放日 | 基金接受申购赎回的日期，通常等于交易日 |

## 份额相关

| 术语 | 含义 |
|------|------|
| A类份额 | 前端收费，申购时扣除申购费 |
| B类份额 | 后端收费，赎回时扣除申购费（逐步淡出） |
| C类份额 | 不收申购费，收取销售服务费（按日计提） |
| 基金份额 | 投资者持有的基金数量单位 |
| 冻结份额 | 因司法冻结等原因不可交易的份额 |
